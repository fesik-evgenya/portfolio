Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности персональных данных (далее - Политика) разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ "О персональных данных" и определяет порядок обработки персональных данных и меры по обеспечению безопасности персональных данных, предпринимаемые Фесик Евгенией (далее - Оператор).

Использование сайта fesik-dev.ru означает безоговорочное согласие пользователя с настоящей Политикой и указанными в ней условиями обработки его персональной информации.

2. Основные понятия

В настоящей Политике используются следующие основные понятия:

  • Персональные данные - любая информация, относящаяся к прямо или косвенно определенному, или определяемому физическому лицу;
  • Обработка персональных данных - любое действие (операция) с персональными данными;
  • Конфиденциальность персональных данных - обязательное для соблюдения требование не допускать их распространения без согласия субъекта;
  • Cookies - небольшие текстовые файлы, размещаемые на устройстве пользователя для анализа поведения на сайте.

3. Состав персональных данных

Оператор может обрабатывать следующие персональные данные пользователей:

  • Фамилия, имя, отчество
  • Электронная почта (e-mail)
  • Номер телефона
  • Компания и должность
  • Сведения о проекте (описание, требования, сроки)
  • Технические данные (IP-адрес, тип браузера, версия ОС)
  • Cookie-файлы и данные аналитики

4. Цели обработки данных

Персональные данные обрабатываются в следующих целях:

  • Обработка запросов и заявок от пользователей
  • Заключение и исполнение договоров на разработку
  • Информирование о новых услугах и акциях
  • Проведение аналитики для улучшения работы сайта
  • Обеспечение безопасности и предотвращение мошенничества

5. Правовые основания обработки

Обработка персональных данных осуществляется на основе:

  • Согласия субъекта персональных данных
  • Необходимости для заключения и исполнения договора
  • Выполнения требований законодательства РФ
  • Законных интересов Оператора

6. Порядок сбора данных

Сбор персональных данных осуществляется следующими способами:

  • Через формы обратной связи на сайте
  • При подписке на email-рассылку
  • Автоматически через системы аналитики (cookies)
  • При заключении договора на оказание услуг

7. Использование файлов cookie

Сайт использует следующие типы cookie-файлов:

  • Обязательные - необходимые для работы основных функций сайта
  • Аналитические - для сбора статистики о посещениях
  • Функциональные - для запоминания пользовательских настроек
  • Рекламные - для показа релевантной рекламы

Пользователь может отключить использование cookies в настройках браузера, однако это может повлиять на работоспособность некоторых функций сайта.

8. Передача данных третьим лицам

Персональные данные пользователей не передаются третьим лицам, за исключением случаев:

  • При получении явного согласия пользователя
  • По требованию уполномоченных государственных органов
  • Для исполнения договорных обязательств
  • При использовании анонимизированных данных для аналитических сервисов

Для обработки данных могут использоваться следующие сервисы:

  • Яндекс.Метрика - веб-аналитика
  • Google Analytics - анализ трафика

9. Защита персональных данных

Оператор предпринимает следующие меры защиты данных:

  • Применение SSL-шифрования для передачи данных
  • Регулярное обновление систем безопасности
  • Ограничение доступа к данным для сотрудников
  • Регулярное резервное копирование информации
  • Использование современных антивирусных систем

Хранение персональных данных осуществляется на территории Российской Федерации.

10. Права субъектов персональных данных

Пользователи имеют право:

  • Получать информацию об обработке своих данных
  • Требовать уточнения, блокирования или уничтожения данных
  • Отозвать согласие на обработку данных
  • Обжаловать действия/бездействие Оператора в уполномоченном органе
